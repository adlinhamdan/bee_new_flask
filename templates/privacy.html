Privacy Policy

This ChatGPT experience, Ground-nesting Bees, was developed by the BeeNews team to help users understand pollinator-friendly practices based on weather and landscape data.

Data Collection

This GPT does not collect, store, or share any personal user data.

Third-Party APIs

We use the Open-Meteo API to fetch weather forecast and historical climate data. The only data sent to the API are location-based coordinates and selected parameters (e.g., temperature, precipitation).

Storage

No user inputs or interactions are stored or logged by the GPT or BeeNews.
